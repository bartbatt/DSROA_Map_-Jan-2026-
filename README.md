# Deer Springs Ranch Property Map

Interactive web map displaying property information for Deer Springs Ranch Owner Association members.

## Features

- **Property Parcels**: All 306 parcels with ownership information
- **Water Infrastructure**: Water mains, irrigation systems, and sampling stations
- **Roads**: BLM Title 5 Rights-of-Way
- **Culverts**: 96 culvert locations (August 2025)
- **Points of Interest**: Key landmarks and facilities
- **Grazing Allotments**: BLM grazing areas

## Usage

Simply open `index.html` in a web browser or host on GitHub Pages.

## Layers

The map includes multiple base layers:
- Satellite imagery (default)
- Topographic map
- Street map

All overlay layers can be toggled on/off using the layer control in the top-right corner.

## Technical Details

- Built with Leaflet.js
- All data in WGS84 (EPSG:4326) projection
- Mobile-responsive design
- GeoJSON data format for fast loading
